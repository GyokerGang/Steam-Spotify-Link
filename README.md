<h1 align="center">Welcome to Steam-Spotify-Link 👋</h1>

#### This project make possible to display on your steam profile your current spotify activity.

## Prerequisites

- npm >=5.5.0
- node >=9.3.0

## Install

```sh
npm install
```

## Usage

```sh
npm run build && npm start
```
